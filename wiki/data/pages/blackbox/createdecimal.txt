=====createdecimal=====
Creates a decimal variable.\\

^Argument ^Description ^
|name|Name of the variable.|
|value|Initial value for the variable|
|min_value|Minimal value allowed to be set for the variable|
|max_value|Maximal value allowed to be set for the variable|
|help|Help shown when using help command on the console|

----
====Returns====
  * Handle to Variable\\

----
====Example====
<Code>
a=blackbox.createdecimal('ratio', 0.5, 0.4, 0.6, 'Conversion ratio')
</Code>
