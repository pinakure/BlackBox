=====ctrlc=====
Useful to debug the code.\\ 
To be called at the end of an user-breakable loop.\\ Once the user has pressed CTRL + C, the function will return **True** once. Subsequent calls will return False unless the user presses CTRL+C again, so this won't work if intending to detect a continous key combination press.

----
====Returns====
True if CTRL+C was pressed

----
====Example====
<Code>
#Will increase by one until CTRL+C is pressed
i=0
while(not blackbox.ctrlc()):
    i=i+1
    console.echo(str(i))
    vpu.update()
</Code>
