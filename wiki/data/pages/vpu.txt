=====VPU=====

----
=== Scripting API ===
^statement           ^arguments                   ^Description                                                                     ^
|[[vpu:createanim  ]]|width, height, sprite_handle|Return Handle to new Animation object                                           | 
|[[vpu:createsprite]]|width, height, filename     |Returns Handle to Sprite object create upon given filename                      | 
|[[vpu:createsurf  ]]|width, height               |Returns Handle to Surface object to be drawn arbitrarily to screen              | 
|[[vpu:deleteanim  ]]|anim_handle                 |Delete Animation identified by given Handle                                     | 
|[[vpu:deletesprite]]|handle                      |Delete Sprite identified by given Handle                                        | 
|[[vpu:deletesurf  ]]|handle                      |Delete Surface identified by given Handle                                       | 
|[[vpu:dimensions  ]]|                            |Returns selected bitmap [ width, height ]                                       | 
|[[vpu:disable     ]]|layer                       |Toggle off given vpu layer                                                      | 
|[[vpu:drawanim    ]]|handle, x, y                |Draw animation identified by given handle onto given coordinates                | 
|[[vpu:drawsprite  ]]|handle, x, y                |Draw Sprite identified by given handle onto given coordinates                   | 
|[[vpu:drawsurf    ]]|handle, x, y                |Draw surface identified by given handle onto given coordinates                  | 
|[[vpu:enable      ]]|layer                       |Toggle on given vpu layer                                                       | 
|[[vpu:fadein      ]]|                            |Fade screen from black                                                          | 
|[[vpu:fadeout     ]]|                            |Fade screen to black                                                            | 
|[[vpu:fading      ]]|                            |Return True is fade in / fade out is activated                                  |
|[[vpu:fill        ]]|r, g, b, a                  |Fill selected bitmap with given color or default color                          |
|[[vpu:fillrect    ]]|x, y, width, height         |Draw a filled rectangle onto selected surface at position x,y.                  |
|[[vpu:frames      ]]|                            |Return actual frame count                                                       | 
|[[vpu:fullscreen  ]]|enabled                     |Toggle fullscreen mode                                                          | 
|[[vpu:rect        ]]|x, y, width, height         |Draw rectangle from (x,y) to (dx,dy) onto selected surface using selected color |
|[[vpu:line        ]]|x, y, dx, dy                |Draw a line onto selected surface from x, y to dx, dy                           |     
|[[vpu:pset        ]]|x, y                        |Draw a pixel onto selected surface onto given coordinates                       | 
|[[vpu:restart     ]]|                            |Restart Video Processing Unit                                                   | 
|[[vpu:rotate      ]]|layer, angle                |Rotate specified layer (0-11) given degrees                                     | 
|[[vpu:select      ]]|layer                       |Select given layer to perform next graphic operations onto it                   | 
|[[vpu:setcolor    ]]|r, g, b, a                  |Sets current painting color                                                     | 
|[[vpu:setfont     ]]|font_name                   |Sets current font                                                               | 
|[[vpu:setrotation ]]|layer, angle                |Sets rotation for specified layer (0-11) at given degrees                       | 
|[[vpu:setscale    ]]|layer, scale_x, scale_y     |Sets scale for specified layer [0-11] given horizontal and vertical values      | 
|[[vpu:scale       ]]|layer, scale_x, scale_y     |Change specified layer [0-11] given horizontal and vertical scale factor        | 
|[[vpu:textout     ]]|text, x, y                  |Print given text at given coordinates                                           | 
|[[vpu:transition  ]]|                            |Start occlusion or dissolution of the dithering transition curtain              | 
|[[vpu:update      ]]|                            |Allow blackbox engine to perform its rendering based input and output operations|
