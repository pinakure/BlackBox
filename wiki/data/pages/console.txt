===== Console =====
----
===Low Level API ===
^command                    ^Description                                                  ^
|[[console:alias          ]]|Set given alias for a command line                           |
|[[console:aliases        ]]|Show list of current aliases                                 |
|[[console:addtoggle      ]]|Adds a toggle with given id and name                         |
|[[console:concat         ]]|Concatenate given arguments and store the value in $result   |
|[[console:cd             ]]|Changes active working directory                             |
|[[console:cls            ]]|Clear console buffer                                         |
|[[console:cwd            ]]|Shows active working directory                               |
|[[console:divide         ]]|Divide given operators and store the value in $result dir    |
|[[console:dir            ]]|Shows files and folders in current working directory         |
|[[console:dump           ]]|Dump console buffer into given filename                      |
|[[console:echo           ]]|Dump given text on the console buffer                        |
|[[console:help           ]]|Show help about requested command                            |
|[[console:multiply       ]]|Multiply given operators and store the value in $result      |
|[[console:memdump        ]]|Dump heap blocks to the filesystem                           |
|[[console:subtract       ]]|Subtract given operators and store the value in $result      |
|[[console:sum            ]]|Sum given operators and store the value in $result           |
|[[console:set            ]]|Set or view value of given variable                          |
|[[console:toggle         ]]|Set or toggle given in-game toggle                           |
|[[console:toggleconsole  ]]|Open or close the console interface                          |
|[[console:togglelist     ]]|Show list of current in-game toggles                         |
|[[console:unalias        ]]|Unset given alias                                            |
|[[console:unset          ]]|Unset given variable                                         |
|[[console:unsetall       ]]|Unsets every variable set                                    |
|[[console:vars           ]]|Show list of defined variables                               |
|[[console:view           ]]|Dump given filename over the console                         |
|[[console:writeconfig    ]]|Stores variable settings under given filename                |
|[[console:xor            ]]|Apply XOR over given value and store value in $result        |
