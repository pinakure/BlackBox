=====deleteanim=====
Delete Animation identified by given Handle\\ 
Internally, this will destroy the animation objects, but it will not delete the sprite data which these point to, therefore you will have to call deletesprite afterwards to completely release its resources, asuming the sprite is not used elsewhere.

^Argument ^Description ^
|anim_handle|Handle identifying target animation to be deleted|

----
====Example====
<Code>
spr = vpu.createsprite('bottle') #Returns sprite handle
ani = vpu.createanim(spr, 0, 0, 1, 1, False)
vpu.deleteanim(spr) #Accepts ani handle
</Code>
