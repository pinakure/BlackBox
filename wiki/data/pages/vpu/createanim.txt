=====createanim=====
Creates an animated sprite object.\\
Requires handle to initialized Sprite.\\ 

^Argument ^Description ^
|width|Width in pixels for the desired animated sprite|
|height|Height in pixels for the desired animated sprite|
|sprite_handle|Sprite handle to be used for frame extraction|

----
====Returns====
  * Handle to Initialized Animation\\
  * Negative on error

----
====Example====
<Code>
a = []
spr = vpu.createsprite('bottle') #Returns sprite handle
a[0] = vpu.createanim(spr, 32, 32, 0, 0, 0, 7, False) #Create animation, each frame 32x32, selecting from 0,0 to 0,7 sweeping horizontally
a[2] = vpu.createanim(spr,  8, 16, 0, 0, 3, 9,  True) #Create animation, each frame  8x16, selecting from 0,0 to 3,9 sweeping vertically</Code>
