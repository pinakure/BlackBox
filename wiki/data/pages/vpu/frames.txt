=====frames=====
Get actual total frame count.\\ This value will not change if the script does not call [[vpu::update|vpu.update()]] periodically.\\ This can be used to perform mildly-precise timing operations when vsync is enabled and there is no frameskipping happening during measurements.

----
====Returns====
Actual frame count

----
====Example====
<Code>
frame = vpu.frames()
while(frame+120 < vpu.frames()):
	#Do stuff...
        #Always call update at least once or loop will be 4ever
	vpu.update() 
</Code>
