=====BlackBox=====

===Scripting API===
^statement                      ^arguments                          ^Description                               ^
|[[blackbox:addtitle          ]]|...                                |Add a game title to the BCH(TM) Dashboard |
|[[blackbox:createbool        ]]|name,value,help                    |Create a boolean variable and get handle  |
|[[blackbox:createdecimal     ]]|name,value,max_value,min_value,help|Create a decimal variable and get handle  |
|[[blackbox:createinteger     ]]|name,value,max_value,min_value,help|Create an integer variable and get handle |
|[[blackbox:createstring      ]]|name,placeholder,max_length,help   |Create a string variable and get handle   |
|[[blackbox:ctrlc             ]]|                                   |Returns TRUE if CTRL+C was pressed        |
|[[blackbox:deletevar         ]]|var_handle                         |Deletes variable by given variable handle |
|[[blackbox:download          ]]|filename                           | |
|[[blackbox:epoch             ]]|                                   |Return current engine epoch uptime        |
|[[blackbox:findvar           ]]|var_name                           |Return handle of var id'd by given name   |
|[[blackbox:getvar            ]]|var_handle                         |Get value of var id'd by given handle     |
|[[blackbox:setvar            ]]|var_handle,value                   |Set value of var id'd by given handle     |
|[[blackbox:version           ]]|                                   |Return current BlackBox engine version    |
